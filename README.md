# DATA-EXPLORATORY-ANALYSIS-FOR-CREDIT-CARD-DATA
This project focuses on analyzing credit card customer data to understand spending behavior, repayment patterns, and overall business performance. Using datasets like customer acquisition, transactions (spend), and repayments, the analysis helps in generating meaningful insights for better decision-making.
The objective is to support the bank in areas such as customer segmentation, fraud detection, profitability analysis, and targeted marketing strategies. By applying data cleaning, transformation, and aggregation techniques, the project uncovers trends in customer activity across different cities, products, and time periods.

Key highlights include:

Data cleaning and preprocessing (handling age, spend, and repayment limits)
Customer behavior analysis (spending vs repayment patterns)
Profit calculation based on monthly interest
Identification of top customers and high-performing product categories
City-wise and time-based analysis with visualizations
Custom Python function to dynamically find top 10 customers by product and time period

This project demonstrates how data analytics can be used in the credit card industry to improve business decisions, reduce risks, and increase revenue.
